# Chat-App
A full-stack mobile chat app built with React Native (Expo) on the frontend and NestJS on the backend. Features include live chatting, searching users, chat history, user authentication (JWT), and profile editing.
# Tech Stack
**Mobile (Frontend)**
- TypeScript / React Native (Expo)
- Mapbox (@rnmapbox/maps) for maps and routing
- React Navigation for screen navigation

**Backend**
- TypeScript / NestJS
- Prisma
- JWT
